## <u>5. HTTP Methods</u>

- [HTTP Methods 참고 링크](https://developer.mozilla.org/ko/docs/Web/HTTP/Methods)

  ![this_screenshot](./img/2.PNG)

  ***

- 데이터 생성(Create) : **POST**
- 데이터 조회(Read) : **GET**
- 데이터 업데이트(Update) : **PUT**
- 데이터 삭제(Delete) : **DELETE**
  - 크롬 - 개발자 도구(Network항목)를 사용하여 Request Method로 확인 가능

## <u>6. POST</u>

- 200번대: 정상적인 결과
  - 200 OK
  - 201 CREATED
- 400번대: 사용자의 잘못된 요청
  - 400 BAD REQUEST
  - 404 NOT FOUND
- 500번대: 서버 내부의 오류
